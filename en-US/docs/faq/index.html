    ProTable request 返回的数据格式可以自定义吗?

    不行的，你可以在 request 中转化一下，或者写个拦截器。

    如何隐藏 ProTable 生成的搜索的 label？

    columns 的 title 支持 function 的，你可以这样写

    title: (_, type) => {
      if (type === 'table') {
        return '标题';
      }
      return null;
    };
